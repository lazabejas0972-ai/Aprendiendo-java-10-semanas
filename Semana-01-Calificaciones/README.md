# Sistema de Calificaciones

## Descripción
Programa en Java que permite ingresar estudiantes y calcular sus promedios.

## Funcionalidades
- Ingreso de datos
- Cálculo de promedio
- Determina aprobado o reprobado
- Estadísticas generales

## Cómo ejecutar
1. Compilar:
   javac Main.java
2. Ejecutar:
   java Main
