# Semana 7 - Agenda Web

Proyecto desarrollado con Java, Maven, Spring Boot y Vaadin.

## Descripción

La aplicación permite navegar entre distintas vistas web usando Vaadin.

Actualmente cuenta con:

- Vista de inicio
- Vista de contactos
- Navegación entre páginas

---

# Tecnologías usadas

- Java 17
- Maven
- Spring Boot
- Vaadin
