# Sistema de Gestión de Vehículos 🚗

## Descripción
Este proyecto es un sistema en Java que permite gestionar una flota de vehículos utilizando Programación Orientada a Objetos. Se implementan conceptos como herencia, polimorfismo e interfaces.


## 🧠 Conceptos aplicados

- Herencia
- Clase abstracta
- Polimorfismo
- Interfaces
- ArrayList

## 📋 Funcionalidades

- Registrar vehículos
- Listar vehículos
- Filtrar por tipo
- Ver vehículos eléctricos
- Cargar baterías
